# 🧪 Python Lab

Este repositório é um espaço de **aprendizado, testes e experimentos com Python**.  
A ideia é pegar conceitos, projetos pequenos ou ideias que vejo por aí e **tentar recriar, testar ou melhorar** usando Python.

Não é um projeto final ou um software específico — é basicamente um **laboratório de código**.

---

## 🎯 Objetivo

- Praticar programação com Python
- Testar ideias que encontro na internet
- Aprender novas bibliotecas
- Melhorar lógica de programação
- Criar pequenos projetos e scripts úteis
